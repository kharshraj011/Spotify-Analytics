# Spotify-Analytics

## About the Project
This project analyzes Spotify data to understand music listening patterns and trends.
It focuses on popular artists, genres, and changes over time.

## What I Analyzed
- Streaming trends
- Popular artists and tracks
- Genre-wise performance

## Tools Used
- Power BI
- Data cleaning
- DAX

## Key Learnings
- A small number of artists get most of the streams
- Some genres are consistently popular
- Listening patterns change over time
